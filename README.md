#  Aprofundamento em JavaScript

Este repositório reúne meus códigos de aprofundamento em **JavaScript**, desenvolvidos durante o curso que realizei com o objetivo de consolidar e expandir meus conhecimentos na linguagem.

##  Objetivo

O foco deste repositório é registrar minha evolução prática em JavaScript, por meio de:

- Exercícios de lógica de programação  
- Manipulação de variáveis, funções e estruturas de controle  
- Arrays e objetos  
- Métodos nativos da linguagem  

---

##  Tecnologias

- JavaScript (ES6+)
- Node.js (quando aplicável)
- HTML5
- CSS3

Os códigos aqui presentes fazem parte do meu processo de aprendizado e aprofundamento na linguagem, servindo como base para projetos futuros e fortalecimento da minha lógica e estruturação de código.
